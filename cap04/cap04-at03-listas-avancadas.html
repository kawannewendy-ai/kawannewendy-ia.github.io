Seção 2 — Ranking de Tecnologias

  1. HTML

    Linguagem responsável pela estrutura das páginas web.

  2. CSS

    Utilizado para estilizar e organizar o visual dos sites.

  3. JavaScript

    Adiciona interatividade e funcionalidades às páginas.

  4. React

    Biblioteca JavaScript muito utilizada no frontend.

  5. Node.js

    Permite executar JavaScript no backend.

Seção 3 — Glossário de Termos Técnicos

HTML
Linguagem de marcação utilizada para criar páginas web.
CSS
Linguagem usada para estilizar páginas HTML.
JavaScript
Linguagem de programação utilizada no desenvolvimento web.
Pode ser utilizada tanto no frontend quanto no backend.
API
Conjunto de regras que permite comunicação entre sistemas.
DOM
Estrutura que representa os elementos da página HTML.
URL
Link
Endereço utilizado para acessar páginas na internet.
HTTP
HTTPS
Protocolos utilizados para comunicação na web.
Frontend
Parte visual de um sistema ou site.
Backend
Parte responsável pelo processamento e banco de dados.
